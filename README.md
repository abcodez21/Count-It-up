# CountItUP
Count It Up assisst users in monitoring caloric intake. 


## Features
* Create an account
* Login to your account
* Reset password
* Add food/meal with (name, calorie count, protein count, carbs count, and fats count)
* Select the date of the meal/food
* Remove food/meal
* Check meal history
